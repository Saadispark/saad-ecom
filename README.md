# Saad E-commerce Template

Next.js + Tailwind eCommerce template with a client-side cart & checkout mock.
Includes:
- pages/index.jsx (catalog)
- pages/checkout.jsx (checkout flow)
- components: Header, ProductCard, CartDrawer, cartContext
- Tailwind setup (styles/globals.css)
- Figma assets and instructions (figma/)



This project is prepared for GitHub. Use Vercel for 1-click deploy.
